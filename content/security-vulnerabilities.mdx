# Security Vulnerabilities

If you discover a security vulnerability 🔓 within **WP Bones** or **WP Kirk**, please send an e-mail to wpbones.info@gmail.com. 

- Alternatively, you can reach out to us on the [Discord](https://discord.gg/5bdVyycU8F) channel.
- Also, you can [submit a security issue](https://github.com/wpbones/WPBones/issues)

🙏 All security vulnerabilities will be promptly addressed.
